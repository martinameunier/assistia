Chapitre 4 - Onglet Agent développeur

Comprendre l'assistant qui peut aider à modifier un projet informatique.

Rôle de l'onglet

Cet onglet est destiné aux personnes qui veulent modifier un projet informatique. L'agent peut lire le dossier configuré, proposer des changements et afficher un résumé.

Si vous ne savez pas ce qu'est un dossier projet ou Git, demandez conseil avant d'utiliser cet onglet. Il peut modifier des fichiers.

Boutons et éléments

StatutIndique si l'agent est prêt, mal configuré ou si des composants manquent.
Installer OllamaApparaît si Ollama manque. Ollama est nécessaire pour que l'agent utilise un modèle local.
Installer l'agentApparaît si Aider n'est pas installé. Il lance l'installation de l'agent développeur.
Bouton ParamètresOuvre l'onglet où renseigner l'URL Ollama, le dossier du projet et le modèle.
Résumé de configurationAffiche l'adresse Ollama, le dossier projet et le modèle utilisés.
Zone de discussionAffiche vos demandes et les réponses de l'agent.
Champ de messageDécrivez ce que vous voulez modifier dans le projet.
Bouton EnvoyerLance la demande auprès de l'agent.
Statut Git et diffSi le projet utilise Git, Assistia peut afficher les changements détectés pour vous aider à vérifier le travail.
