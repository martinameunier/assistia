Chapitre 3 - Onglet Générateur d'image

Créer une image à partir d'une description écrite.

Rôle de l'onglet

Cet onglet crée une image à partir d'une description. Vous n'avez pas besoin de connaître ComfyUI : Assistia le pilote pour vous.

Boutons et éléments

StatutIndique si le générateur d'image est installé, démarré ou absent.
InstallerApparaît si ComfyUI n'est pas installé. Il lance l'installation du générateur d'image.
Liste des modèlesPermet de choisir le modèle d'image à utiliser. Si aucun modèle n'est disponible, téléchargez-en un depuis Paramètres.
Champ PromptÉcrivez ce que l'image doit représenter. Exemple : "un chalet dans la neige au lever du soleil".
Bouton GénérerLance la création de l'image. Pendant la génération, le bouton affiche une progression.
Bouton Ouvrir ComfyUIOuvre l'interface complète de ComfyUI dans le navigateur. Ce bouton est utile pour les utilisateurs avancés.
Aperçu de l'imageAffiche l'image créée dès qu'elle est disponible.
Bouton EnregistrerPermet de choisir où sauvegarder l'image sur votre ordinateur.

Les modèles d'image peuvent être volumineux. Gardez l'application ouverte jusqu'à la fin des téléchargements.
