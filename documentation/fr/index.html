Le livre d'utilisation Assistia

Chaque chapitre explique un onglet de l'application avec des mots simples, pour pouvoir utiliser Assistia sans connaissance informatique.

Sommaire

Lisez les chapitres dans l'ordre pour découvrir l'application, ou ouvrez directement l'onglet qui correspond à votre besoin.

Chapitre 1 Démarrage et installation Comprendre Assistia, le menu, la première installation et les composants. Chapitre 2 Onglet Chat Discuter avec une IA locale, choisir un modèle, utiliser la recherche web et gérer les conversations. Chapitre 3 Onglet Générateur d'image Créer une image, choisir un modèle, lancer la génération et enregistrer le résultat. Chapitre 4 Onglet Agent développeur Comprendre l'assistant de développement et les précautions avant de modifier un projet. Chapitre 5 Onglet Paramètres Installer les composants, télécharger les modèles et configurer l'application. Chapitre 6 Logs, Licence et aide Lire le journal, consulter les licences, ouvrir la documentation et soutenir le projet. Annexe Choisir un modèle Ollama Tableau d'aide pour choisir un modèle selon la puissance de votre ordinateur. Annexe Installation macOS Aide spécifique pour installer et utiliser Assistia sur macOS. Annexe Installation Windows Aide spécifique pour installer et utiliser Assistia sur Windows. Annexe Installation Linux Aide spécifique pour installer et utiliser Assistia sur Linux.
