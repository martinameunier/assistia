Chapitre 1 - Démarrage et installation

Comprendre le rôle d'Assistia, le menu latéral et la première fenêtre d'installation.

À quoi sert Assistia ?

Assistia permet d'utiliser des outils d'intelligence artificielle installés sur votre ordinateur. Vous n'avez pas besoin de taper des commandes : vous cliquez sur des boutons, et Assistia installe, démarre et utilise les composants nécessaires.

ChatDiscute avec un modèle de texte local installé avec Ollama.
Générateur d'imageCrée une image à partir d'une phrase descriptive.
Agent développeurAide à modifier un projet informatique. Cet onglet est surtout destiné aux utilisateurs qui travaillent avec du code.
ParamètresInstalle les composants, télécharge les modèles et règle les options.
LogsAffiche ce qu'Assistia est en train de faire : installation, démarrage, téléchargement ou erreur.

Menu latéral et boutons communs

Le menu à gauche sert à changer d'onglet. Le bouton avec trois lignes permet de réduire ou d'agrandir le menu.

Bouton ChatOuvre la discussion avec l'IA locale.
Bouton Générateur d'imageOuvre l'écran qui permet de créer une image.
Bouton Agent développeurOuvre l'assistant pour les projets de code.
Bouton ParamètresOuvre l'onglet où l'on installe et configure les composants.
Bouton DocumentationOuvre ce guide dans le navigateur.
Bouton LicenceAffiche les licences d'Assistia et des outils utilisés, ainsi qu'un rappel de responsabilité.
Bouton LogsOuvre le journal d'activité.
Bouton PatreonOuvre la page de soutien du projet.
Choix de languePermet de passer l'interface en français ou en anglais.

Première ouverture et installation

Au premier lancement, Assistia peut afficher une fenêtre proposant d'installer les composants nécessaires. Tout est coché par défaut pour simplifier le démarrage.

Cases à cocher des composantsChoisissez ce que vous souhaitez installer. Si vous ne savez pas quoi choisir, laissez tout coché.
OllamaMoteur indispensable pour utiliser le chat et l'agent développeur.
Générateur d'imageInstalle ComfyUI, l'outil utilisé pour créer des images.
Agent développeurInstalle Aider, l'outil utilisé pour aider à modifier des fichiers de projet.
Recherche web SearXNGInstalle un outil local qui permet au chat de trouver des informations récentes sur Internet.
Ne plus afficher automatiquement cette fenêtreCochez cette option si vous ne voulez plus voir cette fenêtre au démarrage.
Installer les composantsLance l'installation des éléments cochés.
Plus tardFerme la fenêtre sans installer. Vous pourrez installer les composants ensuite depuis Paramètres.

Une connexion Internet est nécessaire pour installer les outils et les modèles. Sur macOS et Linux, si curl manque, Assistia tente de l'installer automatiquement.
