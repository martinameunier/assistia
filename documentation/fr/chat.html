Chapitre 2 - Onglet Chat

Discuter avec une IA locale, gérer les conversations et utiliser la recherche web.

Rôle de l'onglet

L'onglet Chat sert à discuter avec un modèle d'IA installé sur votre ordinateur. Il fonctionne avec Ollama.

Boutons et éléments du Chat

Statut en haut de l'ongletIndique si Ollama est prêt, arrêté ou absent. Si Ollama n'est pas installé, le chat ne peut pas répondre.
InstallerApparaît si Ollama manque. Il lance l'installation du composant nécessaire au chat.
Liste des modèlesPermet de choisir l'IA qui répondra. Si la liste est vide, téléchargez un modèle depuis Paramètres.
Indicateur Recherche webMontre si la recherche web est disponible. Elle doit être installée et activée dans Paramètres.
Colonne ConversationsAffiche les anciennes conversations enregistrées sur le disque.
Bouton Nouvelle conversationCrée une discussion vide, sans supprimer les anciennes.
Bouton Supprimer une conversationSupprime la conversation sélectionnée dans l'historique.
Zone des messagesAffiche vos messages et les réponses de l'IA.
Bouton globeActive la recherche web uniquement pour le prochain message. Utilisez-le pour une question qui demande des informations récentes.
Champ de messageÉcrivez ici votre question ou votre demande.
Bouton EnvoyerEnvoie votre message au modèle choisi.
Sources webQuand la recherche web est utilisée, les sources apparaissent dans la réponse. Un clic sur une source l'ouvre dans votre navigateur.

Historique et mot de passe

Assistia peut enregistrer les conversations sur le disque. Si vous définissez un mot de passe dans Paramètres, l'historique est chiffré.

Fenêtre de déverrouillageS'affiche au démarrage si l'historique du chat est protégé par mot de passe.
Champ Mot de passeSaisissez le mot de passe choisi dans Paramètres.
Bouton DéverrouillerOuvre l'historique pour la session actuelle. Le mot de passe reste en mémoire seulement jusqu'à la fermeture de l'application.
Supprimer l'historiqueCe bouton apparaît après trois mots de passe incorrects. Il sert uniquement si vous avez oublié votre mot de passe.
Confirmation de suppressionAvant d'effacer l'historique, Assistia rappelle que l'action est irréversible. Si vous confirmez, les conversations et le mot de passe sont supprimés.

Assistia ne peut pas retrouver un mot de passe oublié. Sans ce mot de passe, la seule solution proposée est la suppression complète de l'historique.
