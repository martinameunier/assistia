Chapitre 5 - Onglet Paramètres

Installer les composants, télécharger les modèles et configurer Assistia.

Installation

Tout installerInstalle tous les composants manquants.
Installer un composantInstalle uniquement la ligne choisie : Ollama, générateur d'image, agent développeur ou recherche web.
Badge Installé / Non installéIndique si le composant est déjà présent sur votre ordinateur.

Modèles du chat

Nom du modèle OllamaChamp où écrire le nom du modèle à télécharger. Une autocomplétion propose des modèles connus, mais vous pouvez saisir un autre nom.
TéléchargerTélécharge le modèle indiqué. Le téléchargement peut être long.
SupprimerEfface un modèle déjà installé pour libérer de l'espace disque.

Pour choisir un modèle adapté à votre machine, consultez le tableau des modèles Ollama.

Modèles du générateur d'image

TéléchargerRécupère les fichiers nécessaires au modèle d'image choisi.
SupprimerSupprime les fichiers du modèle d'image.
Fenêtre de licenceAvant certains téléchargements, Assistia affiche la licence du modèle. Lisez-la avant d'accepter.

Agent développeur

URL OllamaAdresse utilisée par l'agent pour parler à Ollama. La valeur habituelle est http://127.0.0.1:11434.
Dossier du projetDossier dans lequel l'agent peut travailler. Choisissez-le avec soin.
ModèleNom du modèle Ollama utilisé par l'agent.
EnregistrerSauvegarde ces réglages.

Recherche web

Activer la recherche web dans le chatAutorise le chat à utiliser la recherche web lorsque vous activez le bouton globe dans une conversation.
URL SearXNGAdresse du moteur de recherche local. Gardez la valeur proposée si vous n'avez pas de configuration particulière.
Nombre de résultatsDéfinit combien de sources peuvent être transmises au modèle.
Enregistrer la recherche webSauvegarde les options de recherche.

Confidentialité du chat

Nouveau mot de passeDéfinit le mot de passe qui protège l'historique du chat.
Mot de passe actuelDemandé lorsque vous changez un mot de passe déjà défini.
Confirmer le nouveau mot de passeÉvite les erreurs de saisie.
Enregistrer le mot de passeChiffre ou rechiffre les conversations avec le nouveau mot de passe.
