Installer Ollama et Open WebUI sur Linux

Assistia installe et lance les composants IA localement sur votre machine Linux.

1. Installation depuis Assistia

  1. Ouvrez Assistia.
  2. Cliquez sur Paramétrage en bas à droite.
  3. Cliquez sur Installer les composants manquants.
  4. Assistia vérifie Ollama et Open WebUI, puis installe uniquement ce qui manque.

Selon votre distribution, l'installation d'Ollama peut demander des droits administrateur. Suivez les indications affichées par le script officiel si votre terminal ou votre système demande une confirmation.

2. Installation manuelle d'Ollama

Vous pouvez aussi installer Ollama avec la commande officielle :

curl -fsSL https://ollama.com/install.sh | sh

Vérifiez ensuite l'installation :

ollama --version

Si Assistia ne trouve pas Ollama automatiquement, renseignez son chemin dans Paramétrage puis Chemin vers Ollama.

Exemples de chemins
  • Linux/usr/bin/ollama
  • Linux/usr/local/bin/ollama
  • Snap/snap/bin/ollama

3. Open WebUI

Dans la plupart des cas, laissez Chemin vers Open WebUI vide. Assistia utilisera son installation locale gérée.

Si Python 3.11 n'est pas disponible, Assistia installe uv, puis crée un environnement local avec Python 3.11.

Si vous avez installé Open WebUI vous-même, vous pouvez renseigner un chemin comme :

/usr/local/bin/open-webui /usr/bin/open-webui

4. Démarrage

  1. Cliquez sur Démarrer.
  2. Assistia lance Ollama sur 127.0.0.1:11434.
  3. Assistia lance Open WebUI sur 127.0.0.1:8080.
  4. Cliquez sur Ouvrir Open WebUI pour accéder à l'interface.

Sources principales : Ollama, Open WebUI et uv.
