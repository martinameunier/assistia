Installer Assistia et ses composants IA sur Linux

Assistia installe et lance les composants IA localement sur votre machine Linux.

1. Installation depuis Assistia

  1. Ouvrez Assistia.
  2. Cliquez sur Paramétrage en bas à droite.
  3. Cliquez sur Installer les composants manquants.
  4. Assistia vérifie Ollama, ComfyUI et l'agent développeur, puis installe uniquement ce qui manque.

Selon votre distribution, l'installation d'Ollama peut demander des droits administrateur. Suivez les indications affichées par le script officiel si votre terminal ou votre système demande une confirmation.

2. Installation manuelle d'Ollama

Vous pouvez aussi installer Ollama avec la commande officielle :

curl -fsSL https://ollama.com/install.sh | sh

Vérifiez ensuite l'installation :

ollama --version

Si Assistia ne trouve pas Ollama automatiquement, renseignez son chemin dans Paramétrage puis Chemin vers Ollama.

Exemples de chemins
  • Linux/usr/bin/ollama
  • Linux/usr/local/bin/ollama
  • Snap/snap/bin/ollama

3. ComfyUI et agent développeur

Dans la plupart des cas, aucune configuration système supplémentaire n'est nécessaire. Assistia installe ComfyUI et Aider dans ses propres dossiers applicatifs.

Si la version Python nécessaire n'est pas disponible, Assistia installe uv, puis crée l'environnement local adapté au module.

Après installation, téléchargez les modèles depuis Paramètres : modèles Ollama pour le chat et modèles ComfyUI pour la génération d'image.

4. Démarrage

  1. Ouvrez Assistia.
  2. Assistia lance Ollama sur 127.0.0.1:11434 quand le composant est installé mais arrêté.
  3. Si ComfyUI est installé, Assistia tente aussi de démarrer son serveur sur 127.0.0.1:8188.
  4. Utilisez l'onglet Chat pour discuter et l'onglet Générateur d'image pour créer des images.

Sources principales : Ollama, ComfyUI, Aider et uv.
