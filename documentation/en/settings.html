Chapter 5 - Settings tab

Install components, download models, and configure Assistia.

Installation

Install allInstalls all missing components.
Install a componentInstalls only the selected row: Ollama, image generator, developer agent, or web search.
Installed / Not installed badgeShows whether the component is already present on your computer.

Chat models

Ollama model nameField where you type the model to download. Autocomplete suggests known models, but you can enter another name.
DownloadDownloads the specified model. This can take time.
DeleteRemoves an installed model to free disk space.

To choose a model that fits your machine, see the Ollama model table.

Image generator models

DownloadDownloads the files required by the selected image model.
DeleteRemoves image model files.
License windowBefore some downloads, Assistia shows the model license. Read it before accepting.

Developer agent

Ollama URLAddress used by the agent to talk to Ollama. The usual value is http://127.0.0.1:11434.
Project folderFolder where the agent can work. Choose it carefully.
ModelOllama model used by the agent.
SaveSaves these settings.

Web search

Enable web search in chatAllows the chat to use web search when you activate the globe button in a conversation.
SearXNG URLAddress of the local search engine. Keep the proposed value unless you have a special setup.
Number of resultsDefines how many sources can be sent to the model.
Save web searchSaves search options.

Chat privacy

New passwordDefines the password that protects chat history.
Current passwordRequired when changing an existing password.
Confirm new passwordHelps avoid typing mistakes.
Save passwordEncrypts or re-encrypts conversations with the new password.
