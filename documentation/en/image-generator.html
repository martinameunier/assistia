Chapter 3 - Image generator tab

Create an image from a written description.

Role of the tab

This tab creates an image from a description. You do not need to know ComfyUI: Assistia controls it for you.

Buttons and elements

StatusShows whether the image generator is installed, running, or missing.
InstallAppears when ComfyUI is not installed. It starts installing the image generator.
Model listLets you choose the image model to use. If no model is available, download one from Settings.
Prompt fieldWrite what the image should show. Example: "a cabin in the snow at sunrise".
Generate buttonStarts image creation. While generating, the button shows progress.
Open ComfyUI buttonOpens the full ComfyUI interface in the browser. This is mainly useful for advanced users.
Image previewDisplays the created image as soon as it is available.
Save buttonLets you choose where to save the image on your computer.

Image models can be large. Keep the application open until downloads finish.
